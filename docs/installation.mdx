## System Requirements

Before installing PandaAGI SDK, ensure your system meets the following requirements:

<AccordionGroup>
  <Accordion title="Python Version">
    - Python 3.9 or higher
    - Python 3.11+ recommended for best performance
  </Accordion>

  <Accordion title="Operating System">
    - Linux (Ubuntu 18.04+, CentOS 7+)
    - macOS 10.15+
    - Windows 10+ (with WSL2 recommended)
  </Accordion>

  <Accordion title="Dependencies">
    - Docker (for environment isolation)
  </Accordion>
</AccordionGroup>

## Installation Methods

### Using pip (Recommended)

The easiest way to install PandaAGI SDK is using pip:

```bash
pip install panda-agi
```

For the latest development version:

```bash
pip install git+https://github.com/panda-agi/sdk.git
```

### Using uv

For faster installation using uv (a fast Python package installer):

```bash
uv pip install panda-agi
```

Or install uv first if you don't have it:

```bash
# Install uv
curl -LsSf https://astral.sh/uv/install.sh | sh

# Then install panda-agi
uv pip install panda-agi
```

For the latest development version with uv:

```bash
uv pip install git+https://github.com/panda-agi/sdk.git
```

### Using Poetry

If you're using Poetry for dependency management:

```bash
poetry add panda-agi
```

### Using Conda

For Conda users:

```bash
conda install -c conda-forge panda-agi
```

### From Source

For development or the latest features:

<Steps>
  <Step title="Clone the Repository">
    ```bash
    git clone https://github.com/panda-agi/sdk.git
    cd panda-agi-sdk
    ```
  </Step>
  
  <Step title="Install Dependencies">
    ```bash
    pip install -e .
    ```
    
    Or with Poetry:
    ```bash
    poetry install
    ```
  </Step>
  
  <Step title="Verify Installation">
    ```bash
    python -c "import panda_agi; print(panda_agi.__version__)"
    ```
  </Step>
</Steps>

## Docker Setup

For isolated agent environments, install Docker:

<Tabs>
  <Tab title="Ubuntu/Debian">
    ```bash
    # Update package index
    sudo apt-get update
    
    # Install Docker
    sudo apt-get install docker.io
    
    # Start Docker service
    sudo systemctl start docker
    sudo systemctl enable docker
    
    # Add user to docker group
    sudo usermod -aG docker $USER
    ```
  </Tab>
  
  <Tab title="macOS">
    ```bash
    # Install Docker Desktop
    brew install --cask docker
    
    # Or download from https://docker.com/products/docker-desktop
    ```
  </Tab>
  
  <Tab title="Windows">
    ```powershell
    # Install Docker Desktop for Windows
    # Download from https://docker.com/products/docker-desktop
    
    # Or using Chocolatey
    choco install docker-desktop
    ```
  </Tab>
</Tabs>

## API Key Configuration

To use Panda AGI SDK, you need an API key that you can obtain from the PandaAGI platform:

<Steps>
  <Step title="Get Your API Key">
    1. Visit [https://agi.pandas-ai.com/](https://agi.pandas-ai.com/)
    2. Sign in with your GitHub account
    3. Navigate to your dashboard to access your API keys
    4. Copy your `PANDA_AGI_KEY`
  </Step>
  
  <Step title="Set Environment Variable">
    Configure your API key as an environment variable:
    
    <Tabs>
      <Tab title="Linux/macOS">
        ```bash
        export PANDA_AGI_KEY="your-api-key-here"
        ```
        
        To make it persistent, add to your shell profile:
        ```bash
        echo 'export PANDA_AGI_KEY="your-api-key-here"' >> ~/.bashrc
        # or ~/.zshrc if using zsh
        ```
      </Tab>
      
      <Tab title="Windows">
        ```powershell
        # PowerShell
        $env:PANDA_AGI_KEY="your-api-key-here"
        
        # Or set permanently via System Properties
        setx PANDA_AGI_KEY "your-api-key-here"
        ```
      </Tab>
    </Tabs>
  </Step>
  
  <Step title="Verify Configuration">
    Test that your API key is properly configured:
    ```python
    import os
    
    api_key = os.getenv('PANDA_AGI_KEY')
    if api_key:
        print("✅ API key configured successfully")
    else:
        print("❌ API key not found")
    ```
  </Step>
</Steps>

## Environment Variables

Configure your environment with the required API keys:

```bash
# Required: Panda AGI API Key (get from https://agi.pandas-ai.com/)
export PANDA_AGI_KEY="your-panda-agi-api-key"

# Optional: Tavily API Key for enhanced web search functionality
export TAVILY_API_KEY="your-tavily-api-key"
```

<Note>
  **Important**: Keep your API keys secure and never commit them to version control. 
  Consider using environment files (`.env`) or secure key management systems for production use.
</Note>
